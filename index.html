HTML
HyperText Markup Language

It is the standard language used to create and design web pages, defining the structure and content of a webpage by using various tags and elements to format text, images, links, and other multimedia elements for display in web browsers.

CSS
Cascading Style Sheets

It is a stylesheet language used for describing the presentation and visual design of web pages written in HTML. CSS defines how elements on a webpage should be displayed, including aspects like layout, colors, fonts, and spacing, making it an essential component for web page styling and design.

BOOTSTRAP

Bootstrap is a popular open-source front-end framework developed by Twitter. It provides a collection of pre-designed HTML, CSS, and JavaScript components and templates that simplify and expedite web development. Bootstrap helps create responsive and visually appealing websites and web applications with consistent design and functionality across various devices and screen sizes.
